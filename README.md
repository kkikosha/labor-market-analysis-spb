# Labor Market Analysis of Saint Petersburg

## Overview

This project analyzes labor market trends in Saint Petersburg using economic and statistical analysis.

The study focuses on employment structure, wage distribution, and labor demand across different sectors of the economy.

## Goal

The goal of the project is to identify key trends in the labor market and analyze factors influencing employment and wages.

## Methods

The analysis includes:

- statistical analysis of employment data
- analysis of wage distribution
- sectoral comparison
- visualization of labor market trends

## Tools

- Excel
- Statistical analysis
- Data visualization

## Project Structure

report/ — full research report  
data/ — dataset used for analysis  
visuals/ — charts and visualizations  

## Key Findings

The analysis highlights several trends in the labor market:

- differences in wages across sectors
- growth in demand for specific professional groups
- structural changes in employment patterns
